Resumo

Esta política explica como coletamos, usamos e protegemos seus dados pessoais no aplicativo Damião Arte. Respeitamos sua privacidade e seguimos a LGPD (Lei Geral de Proteção de Dados).

1. Quem Somos

O Damião Arte é um aplicativo de galeria digital que permite visualizar, curtir e adquirir obras de arte. Somos responsáveis pelo tratamento dos seus dados pessoais de acordo com esta política.

2. Dados Que Coletamos

2.1 Dados de Cadastro

  • Nome completo: Para identificação e personalização
  • Email: Para login, comunicação e recuperação de senha
  • Senha: Armazenada de forma criptografada para segurança
  • Foto de perfil: Opcional, para personalização da conta

2.2 Dados de Pagamento

  • Nome no cartão: Para processamento de pagamentos
  • CPF: Obrigatório para transações financeiras (BACEN)
  • Telefone: Para contato sobre pedidos e segurança
  • Endereço completo: Para entrega e validação antifraude
  • Dados do cartão: Processados de forma segura pelo Stripe (certificação PCI-DSS Level 1)

2.3 Dados de Uso

  • Obras curtidas: Para personalizar sua experiência
  • Histórico de pedidos: Para acompanhamento e suporte
  • Preferências: Tema, configurações do app
  • Logs de acesso: Para segurança e melhorias

2.4 Dados Técnicos

  • Device ID: Para segurança e prevenção de fraudes
  • Endereço IP: Para localização e segurança
  • Informações do dispositivo: Para compatibilidade

3. Como Usamos Seus Dados

3.1 Finalidades do Tratamento

  • Prestação do serviço: Funcionamento do app e suas funcionalidades
  • Processamento de pagamentos: Transações seguras via Stripe
  • Armazenamento de dados: Informações armazenadas no Firebase (Google) e MongoDB
  • Comunicação: Notificações sobre pedidos e atualizações
  • Segurança: Prevenção de fraudes e proteção da conta
  • Melhorias: Análise de uso para aprimorar o app
  • Cumprimento legal: Obrigações fiscais e regulatórias

3.2 Base Legal (LGPD)

  • Execução de contrato: Dados necessários para o serviço
  • Consentimento: Dados opcionais como foto de perfil
  • Obrigação legal: CPF para transações financeiras
  • Legítimo interesse: Segurança e prevenção de fraudes

4. Compartilhamento de Dados

4.1 Serviços Terceiros

  • Firebase (Google): Autenticação, armazenamento de imagens e dados em tempo real
  • MongoDB Atlas: Armazenamento seguro de dados de pedidos e transações
  • Stripe: Processamento de pagamentos com certificação PCI-DSS Level 1
  • Render.com: Hospedagem segura do backend

4.2 Detalhamento dos Serviços

Firebase (Google Cloud Platform)

  • Autenticação de usuários (Firebase Authentication)
  • Armazenamento de imagens (Firebase Storage)
  • Banco de dados em tempo real (Cloud Firestore)
  • Localização: Servidores nos EUA com conformidade GDPR/LGPD

MongoDB Atlas

  • Armazenamento de dados de pedidos e histórico de transações
  • Backups automáticos criptografados
  • Localização: Servidores com conformidade SOC 2 Type II

Stripe

  • Processamento seguro de pagamentos com cartão de crédito
  • Tokenização de dados de cartão (nunca armazenamos dados completos)
  • Certificação PCI-DSS Level 1 (mais alto nível de segurança)
  • Conformidade com LGPD e regulamentações internacionais

Importante: Nunca vendemos seus dados pessoais. Compartilhamos apenas o necessário para o funcionamento do serviço, sempre com empresas que seguem padrões rigorosos de segurança e conformidade com LGPD.

5. Seus Direitos (LGPD)

Você tem os seguintes direitos sobre seus dados pessoais:

  • Acesso: Saber quais dados temos sobre você
  • Correção: Atualizar dados incorretos ou incompletos
  • Exclusão: Deletar sua conta e dados (disponível no app)
  • Portabilidade: Receber seus dados em formato estruturado
  • Oposição: Contestar o tratamento em certas situações
  • Informação: Saber como seus dados são tratados

5.1 Como Exercer Seus Direitos

  • Exclusão de conta: Disponível no menu Perfil do app
  • Outros direitos: Entre em contato conosco pelos canais informados

6. Segurança dos Dados

6.1 Medidas de Proteção

  • Criptografia: Senhas e dados sensíveis criptografados
  • HTTPS: Comunicação segura entre app e servidor
  • Tokenização: Dados de cartão nunca armazenados diretamente
  • Acesso restrito: Apenas pessoal autorizado acessa os dados
  • Backups seguros: Cópias de segurança criptografadas

6.2 Proteção de Propriedade Intelectual

  • Proteção contra captura: Em dispositivos Android, implementamos proteção contra screenshots e gravação de tela quando você visualiza imagens de obras em tela cheia. Esta medida protege os direitos autorais do artista e a propriedade intelectual das obras
  • Finalidade: Prevenir cópias não autorizadas das obras de arte
  • Transparência: Esta proteção é aplicada apenas durante a visualização de imagens em tela cheia e não afeta outras funcionalidades do app

7. Retenção de Dados

  • Conta ativa: Dados mantidos enquanto a conta existir
  • Após exclusão: Dados removidos em até 30 dias
  • Dados fiscais: Mantidos por 5 anos (obrigação legal)
  • Logs de segurança: Mantidos por 6 meses

8. Dados de Menores

O aplicativo não é destinado a menores de 18 anos. Caso identifiquemos dados de menores coletados sem autorização dos responsáveis, eles serão imediatamente removidos.

9. Cookies e Tecnologias

  • Armazenamento local: Para manter preferências e login
  • Analytics: Para melhorar a experiência do usuário
  • Não utilizamos: Cookies de terceiros para publicidade

10. Transferência Internacional

Alguns dados podem ser processados em servidores fora do Brasil:

  • Firebase (Google): Servidores nos EUA com conformidade GDPR/LGPD
  • MongoDB Atlas: Servidores com conformidade SOC 2 Type II
  • Stripe: Processamento global com conformidade PCI-DSS e LGPD

Todos os provedores seguem padrões internacionais de proteção de dados equivalentes ou superiores à LGPD, incluindo cláusulas contratuais padrão aprovadas e certificações de segurança reconhecidas internacionalmente.
